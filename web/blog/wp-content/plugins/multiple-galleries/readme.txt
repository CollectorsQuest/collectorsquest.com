Create multiple galleries per page/post using the built-in Gallery feature.


== Description ==

This plugin adds checkboxes next to images you have uploaded to a gallery in a post. Selecting checkboxes will include those images in that particular gallery.

To create another gallery within the same post, simply choose 'Insert Gallery' and select the images.


== Instructions ==

1.	Install the plugin.	
2.	Create new gallery, upload photos and select checkboxes next to photos you want to include in that gallery. If no checkboxes are selected, all photos are included.
3.	To create another gallery on the same page, simply click in the area of text editor where you want to place it, and choose photos you want to include.
